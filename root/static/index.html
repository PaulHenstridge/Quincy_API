Quincy API

An API that is worth your time

Endpoints

Get All Links

Returns all the links

/links 

    


    

        
Get a Random Link

        
Returns one link at random

        

               /links/random
        

        

    


    

        
Search Description Text

        
Returns all links where search term is in the description

        

             
             /links/search?term=
        

        

    


    

        
Search by Content Tag

        
Returns  all links where search term is in the list of content tags

        

              /links/search?tag=
        

        

    


    

    

        
Search for Any of Multiple Tags

        
Returns all links where one or more searched term is in the list of content tags

        

              /links/search?tags=
        

        

    


    

        
Search for All of Multiple Tags

        
Returns all links where all searched terms are in the list of content tags

        

              /links/search?tags=
        

        

    


    

    

        
Get All Quotes 

        
Returns a json object containing all the qoutes

        

              /quotes
        

        

    


    

        
Get a Random Quote

        
Returns one quote at random

        

               /links/random
