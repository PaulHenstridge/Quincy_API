Quincy API

An API that is worth your time

Endpoints

Get All Articles

returns a json object containing all the links

/links
Get a Random Article

returns one article at random

/links/random
Search description text

returns a json object containing all links where search term is in the description

/links/search?term=
Search by content tag

returns a json object containing all links where search term is in the list of content tags

/links/search?tag=
Search within content tags

returns a json object containing all links where search term is contained within a content tag

/links/search?term=
Search for any of multiple tags

returns a json object containing all links where one or more searched term is in the list of content tags.

/links/search?tags=
Search for all of multiple tags

returns a json object containing all links where all searched terms are in the list of content tags.

/links/search?tags=
